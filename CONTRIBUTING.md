# Contributing to Giolt

## About AI

AI is helpful, but we prefer code to be written by humans. AI can be used for other purposes such as: 
* creating commit messages
* summarizing pull requests
* autocompletion (small, not 500 lines)
* code scanning for vulnerabilities

We do not allow code that was written entirely by AI. Any pull request detected to have used AI in the 
majority of the code will have the author of the pull request banned.
