> [!WARNING]
> Giolt is still in very early stages of development.

## Development

All of the Giolt source code, from the frontend to the backend, is in a single repository.

To start developing you will need Bun, Node, Erlang, Gleam and Docker installed.

To make development easier, we also added a devcontainer configuration.
